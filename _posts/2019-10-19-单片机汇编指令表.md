| （指令中的符号标识） |                                                         |
| -------------------- | ------------------------------------------------------- |
| Rn                   | 工作寄存器R0-R7                                         |
| Ri                   | 工作寄存器R0 和R1                                       |
| @Ri                  | 间接寻址的8 位RAM 单元地址（00H-FFH）                   |
| #data8               | 8 位常数                                                |
| #data16              | 16 位常数                                               |
| addr16               | 16 位目标地址，能转移或调用到64KROM 的任何地方          |
| addr11               | 11 位目标地址，在下条指令的2K 范围内转移或调用          |
| Rel                  | 8 位偏移量，用于SJMP 和所有条件转移指令，范围-128～+127 |
| Bit                  | 片内RAM 中的可寻址位和SFR   的可寻址位                  |
| Direct               | 直接地址，范围片内RAM 单元（00H-7FH）和80H-FFH          |
| $                    | 指本条指令的起始位置                                    |
